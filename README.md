## What's working:

- Internal graphics Hardware acceleration
- CPU & Fan control
- Wi-Fi 
- USB Ports
- Keyboard
- Trackpad
- Audio / Speaker / Headphone / Microphone
- Screen brightness control via keyboard keys
- Battery Status
- Builtin Camera
- Builtin Card Reader
- VGA output
- HiDPI
- Sleep / Wake up



## What's not working:

- Bluetooth 
- HDMI output
